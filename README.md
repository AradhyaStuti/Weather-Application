
# Weather Application

A clean and modern Weather Application built using **HTML, CSS, and JavaScript**.
It fetches real-time weather data using the **WeatherAPI** and displays temperature, humidity, wind speed, and weather condition with a professional UI.

---

##  Features

*  Search weather by city name
*  Real-time temperature (°C)
*  Humidity details
*  Wind speed information
*  Weather condition with icon
*  Press **Enter key** to search
*  Soft aesthetic pastel gradient UI
*  Responsive design

---

##  Technologies Used

* HTML5
* CSS3 (Modern UI with soft gradients)
* JavaScript (ES6)
* WeatherAPI (Current Weather API)

##  How It Works

1. User enters a city name.
2. App sends a request to WeatherAPI:

   ```
   http://api.weatherapi.com/v1/current.json
   ```
3. The API returns:

   * Location
   * Temperature
   * Weather condition
   * Humidity
   * Wind speed
4. Data is displayed dynamically on the page.

---

##  API Used

This project uses:

**WeatherAPI – Current Weather Endpoint**

To use your own API key:

1. Go to [https://www.weatherapi.com/](https://www.weatherapi.com/)
2. Create a free account
3. Generate your API key
4. Replace this in the code:

```javascript
key=YOUR_API_KEY
```
